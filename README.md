## 🗺️ What Is This Repo?

**Backend Domination** is my personal backend learning playground — a structured, commit-by-commit record of mastering Node.js and backend development from the ground up.

Every folder in this repo represents a **concept learned** or a **mini-project built**. No fluff, no copy-paste — just raw learning in action.

---

## 📂 Repo Structure & Learning Path

```
Backend-Domination/
│
├── 📦 Native_Modules          → Exploring Node.js core (fs, path, os, events)
├── 📦 NPM_Project             → Working with npm, package.json & dependencies
├── 🌐 Express_Node_server     → First Express server — routing, methods & more
├── 📡 HTTP_Requests           → Understanding HTTP verbs (GET, POST, PUT, DELETE)
├── 📬 Postman                 → API testing with Postman
├── 📄 JSON_Intro              → JSON fundamentals — parsing & sending data
├── 📤 Passing_Data            → Passing data via params, query strings & body
│
├── 🖼️ EJS(Embeded-Javascript) → EJS templating engine basics
├── 🏷️ EJS_Tags                → Mastering EJS tags (<%, <%=, <%-, etc.)
├── 🧩 EJS_Partials            → Reusable partials (header, footer, nav)
│
├── 🔗 Axios_Intro             → Making HTTP requests with Axios
├── 🏗️ DIY_API                 → Building your own custom API from scratch
├── 🔐 API_Authentication      → API keys, auth headers & secrets management
├── 🛠️ REST_API_Practice       → Full CRUD with RESTful conventions
├── ⚙️  Middleware              → Custom Express middleware & error handling
│
├── 🎸 Band_Generator_Project  → Fun project: random band name generator
├── 📝 Blog_API_Project        → CRUD blog using REST API
├── 📷 QR-code_generator       → Node.js QR code generator CLI tool
├── 🔑 Secrets_Project         → API auth + secrets handling (v1)
└── 🔑 Secrets_Project-2       → Improved secrets project (v2)
```

## ⚙️ Getting Started

### Prerequisites

Make sure you have these installed:

```bash
node --version    # v18+ recommended
npm --version     # v9+
```

### Clone & Run Any Project

```bash
# 1. Clone the repo
git clone https://github.com/Yuvrajsinghko/Backend-Domination.git

# 2. Navigate to any project folder
cd Backend-Domination/Blog_API_Project

# 3. Install dependencies
npm install

# 4. Run the server
node index.js
# or
nodemon index.js
```

> 💡 Most projects run on **http://localhost:3000**
